## 系统架构

### 前端架构

```
┌─────────────────────────────────┐
│         Web App                 │
│    (Next.js + React)            │
├─────────────────────────────────┤
│         Mobile App              │
│    (React Native + Expo)        │
└─────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────┐
│         API Gateway             │
│    (Next.js API Routes)         │
└─────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────┐
│       Backend Services          │
│  (Supabase Functions + Edge)    │
├─────────────────────────────────┤
│  • 数据采集服务                  │
│  • 数据解析服务                  │
│  • 预警服务                      │
│  • 分析服务                      │
└─────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────┐
│         Database                │
│    (Supabase PostgreSQL)        │
├─────────────────────────────────┤
│  • 用户数据                      │
│  • 股票数据                      │
│  • 财报数据                      │
│  • 预警记录                      │
└─────────────────────────────────┘
```

### 数据流

```
外部数据源 → 数据采集 → 数据解析 → 存储 → API → 前端展示
                 │
                 ▼
           预警检测 → 推送通知
```

## 技术选型

### 后端

| 组件 | 技术 | 说明 |
|------|------|------|
| 数据库 | Supabase PostgreSQL | 关系型数据库，支持 RLS |
| API | Next.js API Routes | 服务端渲染和 API |
| 数据采集 | Python + asyncio | 异步数据采集 |
| 数据解析 | Python + pandas | 财报数据解析 |
| 预警系统 | Supabase Functions | 定时任务触发 |
| AI 分析 | GLM-4 API | 财报摘要和分析 |

### 前端

| 组件 | 技术 | 说明 |
|------|------|------|
| 框架 | Next.js 14 | React 框架 |
| 状态管理 | Zustand | 轻量级状态管理 |
| UI 组件 | shadcn/ui | 基于 Radix UI |
| 图表库 | Recharts / Chart.js | 数据可视化 |
| 表格 | TanStack Table | 高性能表格 |

### 移动端

| 组件 | 技术 | 说明 |
|------|------|------|
| 框架 | React Native | 跨平台 |
| 工具链 | Expo | 开发和构建 |
| 导航 | React Navigation | 路由导航 |
| 推送 | Expo Notifications | 消息推送 |

## 安全策略

1. **数据安全**
   - Supabase Row Level Security (RLS)
   - API 密钥加密存储
   - 用户数据隔离

2. **访问控制**
   - JWT 认证
   - OAuth 登录（Google、Apple）
   - 角色权限管理

3. **速率限制**
   - API 限流
   - 防止滥用
   - 监控异常请求
